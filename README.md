
🌐 موقع تعليم الاختراق الأخلاقي - Ethical Hacking Platform

مرحبًا بك في منصة تعليم الاختراق الأخلاقي، موقع عربي يهدف إلى تقديم محتوى تعليمي مميز في مجال الأمن السيبراني واكتشاف الثغرات بشكل قانوني وأخلاقي.

📌 فكرة المشروع

تم تطوير هذا الموقع بهدف نشر الوعي الأمني وتمكين المهتمين بالمجال من تعلم المهارات اللازمة لفهم حماية الأنظمة، تحليل الثغرات، وتجربة أدوات الهجوم والدفاع السيبراني بشكل عملي وآمن.

✨ مميزات الموقع:
- شروحات مبسطة للمبتدئين والمحترفين.
- دروس مكتوبة ومنظمة حسب المستوى.
- تطبيقات عملية باستخدام أدوات مشهورة مثل Nmap وMetasploit.
- قسم مخصص للهندسة الاجتماعية وتحليل الشبكات.
- واجهة عربية سهلة الاستخدام.


- نظام تسجيل دخول وتسجيل جديد للمستخدمين

📂 طريقة تشغيل المشروع محليًا داخل laravel

```bash
git clone https://github.com/sb3ly/Madar.git
cd Madar-main
copy all files index,...,...,... into laravel project
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```


🎯 أهداف المشروع

- تقديم محتوى تعليمي احترافي باللغة العربية.
- مساعدة الطلاب والمهتمين في الدخول لعالم الأمن السيبراني.
- رفع مستوى الوعي الأمني في العالم العربي.
- تدريب المستخدمين على استخدام أدوات الاختبار الأخلاقي بشكل قانوني وآمن.
